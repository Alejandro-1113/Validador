# Validador de Formulario de Registro

Este proyecto es un formulario de registro con **validación en tiempo real** usando HTML, CSS y JavaScript.  
Valida:

- Nombre (mínimo 3 caracteres)
- Email (formato correcto)
- Contraseña (mínimo 8 caracteres, 1 mayúscula, 1 número)
- Confirmación de contraseña (debe coincidir)

Cada campo muestra **mensajes de error específicos** mientras el usuario escribe.

---

##  Tecnologías usadas
- HTML5  
- CSS3  
- JavaScript (validación en tiempo real)
